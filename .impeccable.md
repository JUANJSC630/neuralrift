# NeuralRift — Design Context

## Project

Spanish-language AI/LLM blog. Guides, reviews, tools, newsletter, affiliate widgets.
Stack: Laravel + Inertia.js + React + Tailwind CSS + Framer Motion.

## Audience

Spanish-speaking developers and entrepreneurs (25–40).
Primary context: desktop, night, deep reading sessions.
Secondary: mobile during commute.

## Brand personality

Sharp, opinionated, credible.
Not: playful, corporate, or generic tech startup.

## Design direction

Keep dark theme (#080B12) and purple/cyan accent system.
Move away from generic AI startup aesthetic toward premium editorial publication.
Less cyber, more authoritative. Think Linear.app meets a high-end magazine.

## Fonts

-   Archivo (display/headings) — variable extended grotesque, inspired by Akzidenz-Grotesk; architectural, precise, authoritative at heavy weights
-   Figtree (body/UI) — variable geometric sans, clean and readable at small sizes
-   JetBrains Mono (code)

## Color system

-   Background: #080B12
-   Accent: #7C6AF7 (purple)
-   Secondary: #06B6D4 (cyan)
-   Gold: #F59E0B
-   Text: #F1F5F9 / #94A3B8

## References — aspire to

Linear.app, Stripe docs, Liveblocks — premium, precise, confident.

## References — avoid

Generic SaaS purple gradient landing pages.
Anything resembling a 2022 AI startup template.
Bounce easing, glow overuse, nested glassmorphism cards.

## Priority pages for design work

1. Blog/Show (post individual) — most important
2. Home hero
3. Blog/Index
4. Admin — secondary priority
